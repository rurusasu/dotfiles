# Tools 設定セクション

OpenClaw のツール設定セクションに関するドキュメント。
各ツールの動作パラメータ、タイムアウト、キャッシュ戦略などを定義する。

## tools.profile

ツールプロファイルを設定する。

| キー      | 値         | 説明                                       |
| --------- | ---------- | ------------------------------------------ |
| `profile` | `"coding"` | コーディング特化のツールセットを有効にする |

利用可能なプロファイル: `coding`, `general` など。

## tools.sessions

セッション間のツール可視性を制御する。

| キー         | 値      | 説明                                       |
| ------------ | ------- | ------------------------------------------ |
| `visibility` | `"all"` | すべてのセッションが互いのツールを参照可能 |

## tools.agentToAgent

エージェント間通信の設定。

| キー      | 値     | 説明                             |
| --------- | ------ | -------------------------------- |
| `enabled` | `true` | エージェント同士の通信を許可する |

## tools.exec

コマンド実行に関するタイムアウトと通知の設定。

| キー                       | 値      | 説明                                             |
| -------------------------- | ------- | ------------------------------------------------ |
| `timeoutSec`               | `120`   | コマンドあたりの最大実行時間（2分）              |
| `backgroundMs`             | `30000` | バックグラウンドに移行するまでの待機時間（30秒） |
| `cleanupMs`                | `5000`  | コマンド終了後のクリーンアップ時間（5秒）        |
| `notifyOnExit`             | `true`  | バックグラウンドコマンド完了時に通知する         |
| `notifyOnExitEmptySuccess` | `false` | 空の正常終了時は通知しない                       |

## tools.web.search

Web 検索機能の設定。

| キー              | 値     | 説明                               |
| ----------------- | ------ | ---------------------------------- |
| `enabled`         | `true` | Web 検索を有効にする               |
| `maxResults`      | `5`    | 検索結果の最大件数                 |
| `timeoutSeconds`  | `30`   | 検索タイムアウト（秒）             |
| `cacheTtlMinutes` | `15`   | 検索結果のキャッシュ有効期間（分） |

## tools.web.fetch

Web ページ取得機能の設定。

| キー              | 値      | 説明                               |
| ----------------- | ------- | ---------------------------------- |
| `enabled`         | `true`  | Web ページ取得を有効にする         |
| `maxChars`        | `50000` | 取得する最大文字数                 |
| `timeoutSeconds`  | `30`    | 取得タイムアウト（秒）             |
| `cacheTtlMinutes` | `15`    | 取得結果のキャッシュ有効期間（分） |

## tools.loopDetection

エージェントが繰り返しパターンに陥ることを防止する設定。

| キー                            | 値     | 説明                               |
| ------------------------------- | ------ | ---------------------------------- |
| `enabled`                       | `true` | ループ検出を有効にする             |
| `historySize`                   | `30`   | 直近のツール呼び出し履歴チェック数 |
| `warningThreshold`              | `10`   | 類似呼び出し10回で警告             |
| `criticalThreshold`             | `20`   | 類似呼び出し20回でクリティカル     |
| `globalCircuitBreakerThreshold` | `30`   | 類似呼び出し30回で停止             |

### detectors

| 検出器                | 状態 | 説明                           |
| --------------------- | ---- | ------------------------------ |
| `genericRepeat`       | 有効 | 汎用的な繰り返しパターンを検出 |
| `knownPollNoProgress` | 有効 | 進捗のないポーリングを検出     |
| `pingPong`            | 有効 | 2つの状態間の往復を検出        |

## 設定の設計根拠

| 設定項目              | 根拠                                                   |
| --------------------- | ------------------------------------------------------ |
| `coding` プロファイル | コード関連ツールを優先的に有効化する                   |
| `exec` タイムアウト   | 暴走コマンドを防止する                                 |
| `backgroundMs=30000`  | 長時間コマンドをブロックせずバックグラウンドに移行する |
| `loopDetection`       | エージェントが繰り返しパターンに陥ることを防止する     |
| Web キャッシュ        | 同一検索に対する API 呼び出しを節約する                |
| `maxChars=50000`      | コンテンツの充実度とトークン消費のバランスを取る       |
