# Issue Templates

This directory contains GitHub issue templates.

## Files

- `bug_report.yml` - Bug report template with structured form
- `feature_request.yml` - Feature request template

## YAML Form Templates

These templates use GitHub's YAML form syntax for structured input, providing a better user experience than markdown templates.

## Customization

Add additional templates as needed:
- `question.yml` - For questions/discussions
- `documentation.yml` - For documentation requests
