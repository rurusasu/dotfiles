# Cursor AI Rules

This directory contains global Cursor AI rules.

## Files

- `rules` - Global Cursor AI rules file

## Usage

These rules apply to all Cursor sessions. For project-specific rules, create a `.cursorrules` file in the project root.

## Deployment

Deployed to `~/.cursor/rules` on all platforms.

## Note

Cursor editor settings (settings.json, keybindings.json) are in `editors/cursor/`.
